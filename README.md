RaspMap
===

Description 
---
RaspMap gather network information on raspbian and report by email
- ifconfig
- get network to scan
- scan bluetooth
- scan Wifi AP
- mtr to google dns
- nmap available network
- nbtscan for gethostnamebyip

Installation 
---
./install 

Usage 
---
Plug the cable and boot ... the email will be sent.
